# Product Roadmap

TerraSense is following a phased approach to ensure product-market fit before massive hardware capital expenditure.

### v0.1: Dashboard Prototype (Current)
* ✅ Define product vision and UI/UX architecture.
* ✅ Build interactive React SPA dashboard.
* ✅ Implement static telemetry visualization (NPK, Moisture, Micro-climate).
* ✅ Design Multispectral Map overlay concept.

### v0.2: Software Intelligence
* ⏳ Transition from static prototype to dynamic dashboard.
* ⏳ Integrate external weather APIs.
* ⏳ Develop initial yield prediction algorithms.
* ⏳ Implement basic Voice Assistant (NLP) functionality for vernacular queries.

### v0.5: Hardware Prototype
* ⏳ Assemble ESP32-based edge node prototype.
* ⏳ Integrate basic sensors (Soil Moisture, DHT22).
* ⏳ Establish local data logging and Wi-Fi transmission.
* ⏳ Connect hardware telemetry directly to the Dashboard.

### v1.0: Pilot Testing
* ⏳ Deploy 10-20 hardware nodes in controlled real-world farming environments.
* ⏳ Test GSM/GPRS connectivity and solar power sustainability.
* ⏳ Gather user feedback on the Voice Assistant accessibility.
* ⏳ Refine AI models based on actual field data.

### v2.0: Commercial Deployment
* ⏳ Mass manufacturing of edge nodes.
* ⏳ Full integration of B2B Data-as-a-Service APIs.
* ⏳ Expansion of vernacular languages supported.
* ⏳ Rollout of premium precision agriculture subscription tier.
